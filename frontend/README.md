## 说明

- 技术栈：Next.js（App Router）+ TypeScript + Tailwind CSS
- 默认开发地址：`http://localhost:3000/`
- 默认后端地址：`http://127.0.0.1:8001`

## 常用命令

```bash
npm install
npm run dev
npm run lint
npm run build
```
