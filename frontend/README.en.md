## Notes

- Stack: Next.js (App Router) + TypeScript + Tailwind CSS
- Default dev URL: `http://localhost:3000/`
- Default backend URL: `http://127.0.0.1:8001`

## Common Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```
