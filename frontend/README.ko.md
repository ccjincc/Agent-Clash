프런트엔드는 `frontend/` 디렉터리에 있습니다. 전체 제품 소개, 아키텍처, 전체 실행 절차는 저장소 루트 README 를 참고하세요.

## 메모

- 스택: Next.js (App Router) + TypeScript + Tailwind CSS
- 기본 개발 URL: `http://localhost:3000/`
- 기본 백엔드 URL: `http://127.0.0.1:8001`

## 자주 쓰는 명령

```bash
npm install
npm run dev
npm run lint
npm run build
```
