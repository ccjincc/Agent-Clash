## Hinweise

- Stack: Next.js (App Router) + TypeScript + Tailwind CSS
- Standard-URL in der Entwicklung: `http://localhost:3000/`
- Standard-Backend-URL: `http://127.0.0.1:8001`

## Haeufige Befehle

```bash
npm install
npm run dev
npm run lint
npm run build
```
