## メモ

- 技術スタック: Next.js (App Router) + TypeScript + Tailwind CSS
- 開発 URL: `http://localhost:3000/`
- 既定のバックエンド URL: `http://127.0.0.1:8001`

## よく使うコマンド

```bash
npm install
npm run dev
npm run lint
npm run build
```
